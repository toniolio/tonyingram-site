# Visual Reference Notes

- The legacy site uses a very minimal personal-site layout with a text wordmark header, a single top-right navigation link, and a simple copyright footer.
- Typography appears close to a system sans stack with bold headings, dark text, and default-style underlined links.
- The home page is structured in three sections:
  - hero with a large photo on the left and headline plus short summary on the right
  - About Tony text block
  - Follow my journey links displayed inline
- The CV page is a long single-column document with narrow content width, standard heading hierarchy, and default-style list spacing.
- Background is white throughout with no decorative chrome beyond spacing and alignment.
